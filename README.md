*   **`Pong-single/`**: A single-player Pong game (vs. AI)
    *   `game.py`: The main game
    *   `arduino/arduino.ino`: The Arduino sketch that reads a single potentiometer
*   **`Pong-duo/`**: A two-player Pong game
    *   `game.py`: The main game
    *   `Pong-duo.ino`: The Arduino sketch that reads two potentiometers
*   **`Safe-Simulation/`**: Paper safe simulatino code for arduino
*       *   `Safe_Simulation.ino`: The Arduino sketch that reads a Servo and a potentiometer
*   **`game-prototype/`**: The main two-player cooperative story game (read more in the folder's README)
